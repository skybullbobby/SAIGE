Usage: step1_fitNULLGLMM.R [options]


Options:
	--plinkFile=PLINKFILE
		Path to plink file for creating the genetic relationship matrix (GRM). minMAFforGRM can be used to specify the minimum MAF of markers in he plink file to be used for constructing GRM. Genetic markers are also randomly selected from the plink file to estimate the variance ratios

	--phenoFile=PHENOFILE
		Path to the phenotype file. The file can be either tab or space delimited. The phenotype file has a header and contains at least two columns. One column is for phentoype and the other column is for sample IDs. Additional columns can be included in the phenotype file for covariates in the null GLMM. Please note that covariates to be used in the NULL GLMM need to specified using the argument covarColList

	--phenoCol=PHENOCOL
		Coloumn name for phenotype to be tested in the phenotype file, e.g CAD

	--traitType=TRAITTYPE
		binary/quantitative [default=binary]

	--invNormalize=INVNORMALIZE
		if quantitative, whether asking SAIGE to perform the inverse normalization for the phenotype [default='FALSE']

	--covarColList=COVARCOLLIST
		List of covariates (comma separated)

	--sampleIDColinphenoFile=SAMPLEIDCOLINPHENOFILE
		Column name of sample IDs in the phenotype file, e.g. IID

	--tol=TOL
		Tolerance for fitting the null GLMM to converge [default=0.02].

	--maxiter=MAXITER
		Maximum number of iterations used to fit the null GLMM [default=20].

	--tolPCG=TOLPCG
		Tolerance for PCG to converge [default=1e-5].

	--maxiterPCG=MAXITERPCG
		Maximum number of iterations for PCG [default=500].

	--nThreads=NTHREADS
		Number of threads (CPUs) to use [default=1].

	--SPAcutoff=SPACUTOFF
		Cutoff for the deviation of score test statistics from mean in the unit of sd to perform SPA [default=2].

	--numRandomMarkerforVarianceRatio=NUMRANDOMMARKERFORVARIANCERATIO
		An integer greater than 0. Number of markers to be randomly selected for estimating the variance ratio. The number will be automatically added by 10 untial the coefficient of variantion (CV) for the variance ratio estimate is below ratioCVcutoff [default=30].

	--skipModelFitting=SKIPMODELFITTING
		Whether to skip model fitting and only to estimate the variance ratio. If TRUE, the file outputPrefix.rda is required [default='FALSE']

	--memoryChunk=MEMORYCHUNK
		Size (Gb) for each memory chunk [default=2]

	--tauInit=TAUINIT
		Initial values for tau. [default=0,0]

	--LOCO=LOCO
		Whether to apply the leave-one-chromosome-out (LOCO) approach. This option has not been extensively tested [default=FALSE].

	--traceCVcutoff=TRACECVCUTOFF
		Threshold for coefficient of variation (CV) for the trace estimator. Number of runs for trace estimation will be increased until the CV is below the threshold [default=0.0025].

	--ratioCVcutoff=RATIOCVCUTOFF
		Threshold for coefficient of variation (CV) for estimating the variance ratio. The number of randomly selected markers will be increased until the CV is below the threshold [default=0.001]

	--outputPrefix=OUTPUTPREFIX
		Path and prefix of the output files [default='~/']

	--outputPrefix_varRatio=OUTPUTPREFIX_VARRATIO
		Path and prefix of the output the variance ratio file [default=NULL]. if NULL, it will be the same as the outputPrefix

	--IsSparseKin=ISSPARSEKIN
		Whether to use sparse kinship for association test [default='FALSE']

	--sparseGRMFile=SPARSEGRMFILE
		Path to the pre-calculated sparse GRM file. If not specified and  IsSparseKin=TRUE, sparse GRM will be computed [default=NULL]

	--sparseGRMSampleIDFile=SPARSEGRMSAMPLEIDFILE
		Path to the sample ID file for the pre-calculated sparse GRM. No header is included. The order of sample IDs is corresponding to sample IDs in the sparse GRM [default=NULL]

	--numRandomMarkerforSparseKin=NUMRANDOMMARKERFORSPARSEKIN
		Number of randomly selected markers to be used to identify related samples for sparse GRM [default=2000]

	--isCateVarianceRatio=ISCATEVARIANCERATIO
		Whether to estimate variance ratio based on different MAC categories. If yes, variance ratio will be estiamted for multiple MAC categories corresponding to cateVarRatioMinMACVecExclude and cateVarRatioMaxMACVecInclude. Currently, if isCateVarianceRatio=TRUE, then LOCO=FALSE [default=FALSE]

	--relatednessCutoff=RELATEDNESSCUTOFF
		Threshold to treat two samples as unrelated if IsSparseKin is TRUE [default=0.125]

	--cateVarRatioMinMACVecExclude=CATEVARRATIOMINMACVECEXCLUDE
		vector of float. Lower bound of MAC for MAC categories. The length equals to the number of MAC categories for variance ratio estimation. [default='0.5,1.5,2.5,3.5,4.5,5.5,10.5,20.5']

	--cateVarRatioMaxMACVecInclude=CATEVARRATIOMAXMACVECINCLUDE
		vector of float. Higher bound of MAC for MAC categories. The length equals to the number of MAC categories for variance ratio estimation minus 1. [default='1.5,2.5,3.5,4.5,5.5,10.5,20.5']

	--isCovariateTransform=ISCOVARIATETRANSFORM
		Whether use qr transformation on non-genetic covariates [default='TRUE'].

	--isDiagofKinSetAsOne=ISDIAGOFKINSETASONE
		Whether to set the diagnal elements in GRM to be 1 [default='FALSE'].

	--useSparseSigmaConditionerforPCG=USESPARSESIGMACONDITIONERFORPCG
		Whether to sparse GRM to speed up the PCG. Current this option is deactivated. [default='FALSE'].

	--useSparseSigmaforInitTau=USESPARSESIGMAFORINITTAU
		Whether to use sparse Sigma to estiamte initial tau [default='FALSE'].

	--minMAFforGRM=MINMAFFORGRM
		Minimum MAF of markers used for GRM

	--minCovariateCount=MINCOVARIATECOUNT
		If binary covariates have a count less than this, they will be excluded from the model to avoid convergence issues [default=-1] (no covariates will be excluded).
